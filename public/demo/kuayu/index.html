iframe操作

  • 父窗口获取子窗口的window对象方法

    Iframe.contentWindow

  • 父窗口获取子窗口document对象方法

    iframe.contentWindow.document || iframe.contentDocument

  • 子窗口获取父窗口window对象

    parent

  • iframe父窗口和子窗口通信符合同源策略

    即同域名下才可以进行window和dom操作

document.domain

原理：

  • 化跨域通信问题为frame之间的跨域操作

  • 两个不同子域名通过设置 document.domain='主域名' 获取操作对方的跨域权限

document.domain跨域实例

  • http://www.xitao.com/demo/kuayu/demo1.html

    document.domain = 'xitao.com';
var iFrame = document.getElementsByTagName("iframe")[0];
document.getElementById('frame-content').innerHTML = "iframe的链接是："+iFrame.contentWindow.location.href;

  • http://other.xitao.com/demo/kuayu/request.html

    document.domain = 'xitao.com';

  • 查看demo

优点

  • 子域名双向跨域

  • post/get皆可

  • 跟本域操作无区别

缺点

  • 安全！！！

  • 不能跨主域名

location.hash

原理：

  • hash指的是url中#之后的部分，跨域时通过iframe.src设置，子窗口通过location.hash取到

  • 设置url的hash不需要页面跳转，而是当前页面锚点跳转

postMessage

原理：

  • html5新增的postMessage方法，通过postMessage来传递信息，对方可以通过监听message事件来监听信息

postMessage浏览器支持情况

PostMessage跨域实例

  • http://www.xitao.com/demo/kuayu/demo5.html

    window.onload = function(){
var iFrame = document.getElementsByTagName("iframe")[0].contentWindow;
iFrame.postMessage('i am demo5','http://other.xitao.com');//第一个参数即你要发送的数据，第二个参数是非常重要，主要是出于安全的考虑，一般填写允许通信的域名

var onmessage = function(e) { 
   var data = e.data;
   alert(data);
};  
//监听postMessage消息事件 
if (typeof window.addEventListener != 'undefined') { 
  window.addEventListener('message', onmessage, false); 
} else if (typeof window.attachEvent != 'undefined') { 
  window.attachEvent('onmessage', onmessage); 
} 
}

其他跨域解决方法

  • flash

    利用flash的URLLoader，也可以轻松实现跨域数据交互

    只要站点B的跨域策略文件(crossdomain.xml)中包含了站点A

    Flash提供ExternalInterface接口与JavaScript通信，ExternalInterface有两个方法，call和addCallback，call的作用是让Flash调用js里的方法，addCallback是用来注册flash函数让js调用

    缺点：需要浏览器flash插件、开发成本大

  • 后端代理

    跨域时，前端浏览器向同一域名下的proxy发出请求，要求获取其它域名下的数据（这个请求不跨域），后端再据此对指定地址发出跨域请求（后端curl不受跨域限制），收到响应后后端把响应体重新包装、返回给前端。

  • 跨域资源共享（CORS）

    与普通AJAX相同，需后端设置Access-Control-Allow-Origin头，而且低版本IE不支持，参考文献。
